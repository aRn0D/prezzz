Let's design our rich model

						
namespace Al\Domain;

final class Employee implements EmployeeInterface,
{
	// properties

	private function __construct(Uuid $identifier, string $name, string $position, \DateTime $hiredAt)
	{
		$this->id = $identifier;
		$this->name = $name;
		$this->position = $position;
		$this->hireAt = $hiredAt
		$this->firedAt = null;
	}

	public static function hire(Uuid $identifier, string $name, string $forPosition, \DateTime $hiredAt)
	{
		return new self($identifier, $name, $forPosition, $withSalaryScale, $hiredAt);
	}

	public function promote(string $toNewPosition)
	{
		$this->position = $toNewPosition;
	}

	public function fire(\DateTime $firedAt)
	{
		$this->firedAt = $firedAt;
	}
}

First problem: Do we use Doctrine correctly?

What is a repository?

Mediates between the domain and data mapping layers using a collection-like interface for accessing domain objects

Operations performed over them provide a more object-oriented view of the persistence layer

What does it look like?

						
namespace Al\Infrastructure;

final class EmployeeRepository implements EmployeeRepositoryInterface
{
	// Constructor and property

	public function get(Uuid $identifier)
	{
		if (null === $employee = $this->entityManager->find(Employee::class, $identifier->toString())) {
			throw new NotExistingEmployee((string) $identifier->toString());
		}

		return $employee;
	}

	public function add(EmployeeInterface $employee)
	{
		$this->entityManager->persist($employee);
		$this->entityManager->flush();
	}

	public function remove(EmployeeInterface $employee)
	{
		$this->entityManager->remove($employee);
		$this->entityManager->flush();
	}
}

Command pattern to the rescue!

A Command is an object that represents all the information needed to call a method.

Let’s create a command

						
namespace Al\Application;

final class HireEmployeeCommand
{
	/** @var string */
	private $name = '';

	/** @var string */
	private $position = '';

    public function getName(): string
    {
        return $this->name;
    }

    public function setName(string $name)
    {
        $this->name = $name;
    }

    public function getPosition(): string
    {
        return $this->position;
    }

    public function setPosition(string $position)
    {
        $this->position = $position;
    }
}

Let’s update our controller

						
namespace Al\Presenter\Controller;

class EmployeeController extends Controller
{
	public function hireAction(Request $request)
	{
		$form = $this->createForm(EmployeeType::class, new HireEmployeeCommand());
		$form->handleRequest($request);

		if ($form->isSubmitted() && $form->isValid()) {
			$employee = $form->getData();

			Employee::hire(Uuid::uuid4(), $employee->getName(), $employee->getPosition());
			$this->get('employee.repository')->add($employee);

			return $this->redirectToRoute('employee_list');
		}

		return $this->render('employee/hire.html.twig', [
			'form' => $form->createView()
		]);
	}
}

Command Bus to the rescue

A Command Bus accepts a Command object and delegates it to a Command Handler.

Let’s create a command handler

						
namespace Al\Application\Handler;

final class HireEmployeeHandler
{
    // Constructor and properties

    public function handle(HireEmployee $command)
    {
        $employee = Employee::hire(
            $command->getName(),
            $command->getPosition(),
            //$command->joinCompanyAt() // TODO
        );

        $this->employeeRepository->add($employee);
    }
}

Let’s update our controller

						
namespace Al\Presenter\Controller;

class EmployeeController extends Controller
{
	public function hireAction(Request $request)
	{
		$form = $this->createForm(EmployeeType::class, new HireEmployeeCommand());
		$form->handleRequest($request);

		if ($form->isSubmitted() && $form->isValid()) {
            $employeeCommand = $form->getData();

            try {
                $this->get('command_bus')->handle($employeeCommand);
            } catch (\Exception $e) {
                $this->addFlash('negative', 'An error occurs, please contact an administrator');
            }

			return $this->redirectToRoute('employee_list');
		}

		return $this->render('employee/hire.html.twig', [
			'form' => $form->createView()
		]);
	}
}

		// Set value to properties
